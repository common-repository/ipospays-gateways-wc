== Description ==

A powerful WooCommerce gateway plugin allowing you to accept major credit cards, ACH, and alternative payment methods with ease.

= Visit our demo site to see all the payment methods in action =
[Demo Site](https://docs.ipospays.tech/product/wooden-bed/)

To use Google Pay, visit the site using a supported browser like Chrome. To use Apple Pay (Coming soon), visit the site using an iOS device.

= Features: =

* Credit/Debit Cards
* Google Pay
* PayPal
* Paylater
* Pockyt
* Venmo
* Crypto
* ACH Payments
* Alternate Payments

== Google Pay Support ==

iPOSpays includes **Google Pay** support, allowing customers to quickly and securely pay using payment details saved to their **Google Account**. With **Google Pay**, checkout is as easy as a single tap on **mobile** and **desktop** devices. 

- **Google Pay**: For more information regarding data handling and user privacy, please refer to the following:
  - [Google Pay Privacy Policy](https://payments.google.com/payments/apis-secure/get_legal_document?ldo=0&ldt=privacynotice&ldl=en-GB)
  - [Google Pay Terms of Service](https://payments.google.com/payments/apis-secure/get_legal_document?ldt=googlepaytos)

== Frequently Asked Questions ==

= How do I test this plugin? =

You can enable the plugin’s test mode, which allows you to simulate transactions.

= Does your plugin support WooCommerce Subscriptions? =

No, our plugin currently does not support WooCommerce Subscriptions. While we understand the importance of this feature, it’s not currently part of our plugin’s functionality. We’re always working on improving our plugin and may consider adding support for WooCommerce Subscriptions in the future.

= Where can I find documentation? =

You can find our detailed documentation [here](https://docs.ipospays.com/iPOSpays-WooCommerce-Extension).

= Where can I get support or talk to other users? =

If you get stuck, you can ask for help in the Plugin Forum.
